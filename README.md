# Machine Learning Tools for Simulations in Electrocatalysis - Tutorial Notebooks

This repository accompanies the article: 

Bruno Inácio da Silva Roux Leite, Anna Stepanova, Timmo-Hendrik Pukk, Denis Savchenko, Thor Kongstad Madsen, Nadezda Kongi, Vladislav Ivanistsev. (2026). Machine Learning Tools for Simulations in Electrocatalysis. ACS Materials Au.
DOI: xxx

## Description

This repository contains tutorial-style Jupyter notebooks and environment configuration files (apt.txt, environment.yaml, postBuild) designed for execution via mybinder.org

The materials demonstrate how machine learning (ML) methods can be integrated into electrocatalysis workflows alongside density functional theory (DFT). Using the oxygen reduction reaction on platinum-based catalysts as a model system, the notebooks implement seven ML approaches, including machine-learned exchange–correlation functionals, Gaussian process optimization, Bayesian configuration search, and ML-based interatomic potentials for geometry optimization and molecular dynamics at near-DFT accuracy. Each notebook provides a reproducible, chemically meaningful example, illustrating both the practical advantages and current limitations of ML in catalytic simulations, as well as its integration into standard DFT-based computational workflows.
